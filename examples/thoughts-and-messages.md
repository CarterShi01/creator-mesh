# Thoughts and Messages

Examples of the two core input primitives that CreatorMesh processes.

## Thoughts

Thoughts are internally generated inputs.

Examples:

- A new product idea
- A personal reflection
- A note from reading a book
- A business insight
- A career plan
- A creative direction
- A decision that needs further thinking
- A draft concept for a future project

## Messages

Messages are externally triggered inputs.

Examples:

- A user feedback message
- A client request
- An investor reply
- A job opportunity
- A collaboration proposal
- A task from a work conversation
- A comment from a community member
- A message that requires a response or follow-up

## Output Possibilities

Thoughts and messages can be transformed into:

- Structured knowledge
- Plans
- Actions
- Workflows
- Shipped products

Human review and approval should be supported throughout the transformation process.
