# events/

Runtime event type and JSONL writer.

- `runtime-event.ts` — `RuntimeEvent` interface
- `runtime-event-writer.ts` — appends JSONL to `~/creator-mesh-runtime/runtime-events/runtime-events.jsonl` (configurable via `CREATORMESH_EVENTS_DIR`)

Events support: debugging, visualization, replay, workflow extraction, audit, governance, and knowledge distillation.
